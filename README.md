# ChaTODO LIST 

## 👨‍🏫 프로젝트 소개
React를 사용하여 만든 **TODO LIST** 입니다. 

## 📢 Running
```js
#development
~$ yarn

# watch mode
~$ yarn dev
```

## 📌 주요 기능

- TODO **추가**
- TODO **삭제**
- TODO 완료버튼 클릭 시 Done 파트로 이동하며 “완료” 텍스트가 ”취소” 텍스트로 변경
- Done에서 “취소” 버튼을 클릭시 Working 파트로 카드이동되며 “완료” 텍스트로 변경
